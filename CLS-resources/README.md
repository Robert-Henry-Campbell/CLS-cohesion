# CLS-resources

Reference material for analysing the Community Life Survey (CLS) Secure Access microdata in the UK Data Service (UKDS) SecureLab. This folder is excluded from version control via `.gitignore`.

Compiled 2026-07-20 from UK Data Service and GOV.UK sources. The application process is out of scope; all files concern analysis after approval.

## Contents

| File | Coverage |
|---|---|
| `01-cls-datasets.md` | UKDS catalogue entries: Secure Access study (SN 9515), End User Licence studies, series information, survey status |
| `02-cls-documentation.md` | Questionnaires, technical reports, derived variables, data tables, weighting variables |
| `03-securelab-environment.md` | Access routes, software (including R), package installation, drives, connectivity constraints |
| `04-outputs-and-sdc.md` | Output release process, minimum requirements, statistical disclosure control rules and guidance |
| `05-import-support-contacts.md` | Importing syntax/files (ZendTo), project changes, training, support contacts, forms |
