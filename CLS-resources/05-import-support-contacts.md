# Importing files, project administration, support

## Importing syntax and files into SecureLab

Files enter SecureLab through UKDS staff via the ZendTo secure transfer service (drop-box style upload, up to 2 GB per request), not email attachment. Procedure (SecureLab User Guide, section 11):

1. Complete the relevant request form; where required UKDS forwards it to the data owner for approval.
2. Send the file (e.g. R script, Stata do-file, SPSS syntax) via ZendTo, including your name and project number.
3. UKDS checks the file and places it in your project area.

Scan files for viruses before sending. Importing external datasets additionally requires evidence of permission to use them in SecureLab. This import route is how code developed against synthetic data is moved on-TRE.
